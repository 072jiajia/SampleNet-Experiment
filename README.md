## Docker Environment

```bash
docker pull asafmanor/pytorch:samplenetreg_torch1.4
docker run --runtime nvidia -v $(pwd):/workspace/ -it --name {your docker's name} asafmanor/pytorch:samplenetreg_torch1.4
```
